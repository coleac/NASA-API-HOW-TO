How-To Guide NASA APIs

The NASA API portal provides one-stop access to over a dozen APIs. These APIs allow users access to data and images available in NASA databases.

We will be taking a look at three of these APIs: the EPIC API, the CAD API, and the Sentry API. This guide is intended for beginners with some background in JavaScript.

Getting Started

The first step we will want to take is to sign up for an API key. This is not required, as one can use the DEMO_KEY, but that will only allow 50 requests/day whereas having an API key allows 1000 requests/hour. The NASA API site has a very easy to follow Getting Started page. The API request only requires your name and email address. Go there now, and fill out the form.

While you were there, did you check out the Live Editor example that demonstrates using the APOD (Astronomy Picture Of the Day) API? If you missed it, go back and take a look. Otherwise, scroll down, and let's put that API key to work!

EPIC API

The DeepSpace Climate Observatory (DSCOVR) is a National Oceanic and Atmospheric Administration (NOAA) Earth observation and space weather satellite launched in February 2015. It carries the Earth Polychromatic Imaging Camera (EPIC) that takes breathtaking images of the Earth. These images have been made available on the EPIC website since October 2015. The images and their metadata can also be accessed through the EPIC API.

CAD API

The Center for Near-Earth Object Studies (CNEOS) is NASA's center for computing asteroid and comet orbits, and their odds of Earth impact. The Near-Earth Object Close-Approach Data (CAD) API gives one access to the Jet Propulsion Lab's (JPL) Small-Body DataBase (SBDB).

Sentry API

Sentry is our Earth impact monitoring system. According to NASA's JPL/CNEOS Sentry website, "Sentry is a highly automated collision monitoring system that continually scans the most current asteroid catalog for future impact with Earth over the next 100 years." The Sentry API provides one access to the data this system collects.
